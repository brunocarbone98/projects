01
DEVSECOPS · MONITOREO DE SEGURIDAD

CyberShield

Un backend de microservicios con la seguridad integrada en cada etapa del ciclo de desarrollo, desde el código hasta la infraestructura en producción.

En qué consiste

Un backend de microservicios automatizado, construido sobre un pipeline de CI/CD que ejecuta análisis estático de código y pruebas de API en cada cambio. La infraestructura está containerizada y es escalable, con detección de intrusiones en tiempo real mediante alertas y registro centralizado de toda la actividad del sistema.

Propósito

Demostrar prácticas DevSecOps: integrar la seguridad y la calidad del código dentro del propio flujo de desarrollo, en lugar de tratarlas como un paso final. El sistema valida automáticamente cada cambio antes de desplegarlo y vigila la infraestructura en producción, reflejando la capacidad de construir software seguro, observable y listo para escalar.

02
FULL-STACK · ARQUITECTURA DISTRIBUIDA

Full-Stack Hub

Una interfaz reactiva y multilingüe sobre una arquitectura distribuida que conecta varios lenguajes y servicios en un solo sistema.

En qué consiste

Una interfaz web reactiva y optimizada para SEO, respaldada por una arquitectura distribuida: una API transaccional en Node.js conectada a una base de datos PostgreSQL, junto a microservicios escritos en Python. El frontend y el backend trabajan en conjunto como un único hub con soporte multilingüe.

Propósito

Demostrar capacidad full-stack de punta a punta: diseñar un frontend moderno y rápido y, a la vez, orquestar un backend distribuido donde distintos lenguajes y servicios (Node.js y Python) se comunican y comparten persistencia. Refleja el manejo de arquitecturas reales de producción, no de un sitio estático.

03
FRONTEND · WEB MULTILINGÜE

Portafolio Web Personal

Este mismo sitio: un portafolio de dos páginas, traducido a nueve idiomas y construido desde cero sin frameworks, solo con tecnologías web puras.

En qué consiste

Un portafolio personal compuesto por dos páginas conectadas: la página principal —con mi presentación, experiencia, habilidades y blog— y esta página de proyectos. Todo está construido desde cero con HTML, CSS y JavaScript puro, e incluye un sistema de internacionalización propio con nueve idiomas sincronizado entre ambas páginas, animaciones de aparición y de escritura, un fondo que cambia sutilmente de color y un diseño totalmente responsivo.

Propósito

Demostrar dominio de los fundamentos del frontend sin depender de frameworks: construir una experiencia interactiva, multilingüe y pulida usando solo tecnologías web nativas. Además, es mi carta de presentación en acción: el proyecto que estás evaluando es la misma página en la que lo estás leyendo.
